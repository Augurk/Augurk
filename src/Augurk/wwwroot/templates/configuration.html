 Show feature dependencies (Alpha)

When working with features, you hardly ever describe a feature that is designed to work in a vacuum. Most features are build on top of other features, or are themselves the foundation for other features. Augurk can detect these inter-feature dependencies and will display them along with the feature descriptions to help create a better understanding of the feature within its full context.

Instuctions: Additional data is required to analyze the dependencies within your features. This data can be collected using the Augurk Analyzers.
At this moment there is only a C# version available, please visit its projectpage for instructions on how to collect the data: linkje

This feature is currently in Alpha. As of yet, there is no formal support for this feature, but any and all feedback is much appreciated.

Integration

Augurk exposes several APIs to enable integration in your own tools: Go to documentation.

Of course, the easiest way to interact with the Augurk APIs is through the commandline tool which is available on NuGet: https://www.nuget.org/packages/Augurk.CommandLine/.

Users of Microsoft ® Visual Studio Team Services or Microsoft ® Team Foundation Server can also use the Augurk build integration: https://marketplace.visualstudio.com/items?itemName=augurk.augurk.
